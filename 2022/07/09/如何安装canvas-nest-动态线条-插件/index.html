如何安装canvas-nest(动态线条)插件

发表于 2022-07-09

hexo博客框架如何安装canvas-nest(动态线条)插件(这里我以next主题演示其他主题大同小异)

添加动态线条

1.添加动态背景

blog更目录打开cmd,输入

git clone https://github.com/theme-next/theme-next/theme-next-canvas-nest

themes/next/source/lib/canvas-nest

2.增加波浪背景动画

继续输入

git clone https://github.com/theme-next/theme-next-three themes/next/source/lib/three

3.配置

\blog\themes\next_config.yml(配置详见： https://github.com/hustcc/canvas-nest.js#configuration)

该文件下添加代码

canvas_nest:
enable: true
color: ‘0, 0, 0’ # 线条颜色，默认：'0,0,0'；三个数字分别为(R,G,B)，注意用，分割
pointColor: ‘156,39,176’ # 交点颜色，默认：'0,0,0'；三个数字分别为(R,G,B)，注意用，分割
opacity: ‘0.8’ # 线条透明度（0~1）, 默认: 0.5
count: ‘99’ # 线条的总数量, 默认:99
zIndex: ‘-1’ # 背景的z-index属性，css属性用于控制所在层的位置, 默认: -1
